CONCEPT

롯데그룹의 식품 계열사인 롯데칠성음료를 벤치마킹 했습니다. 기존 사이트의 독특한 텍스트 효과와 배경에 들어가있는 동영상 효과를 적용해보았습니다. 대표적인 효과로는 스크롤 시 각 섹션의 타이틀을 천천히 사라지게 하는 효과를 구현하였고, 섹션의 배경에 청량감을 줄 수 있는 동영상을 넣어 대표 제품을 강조하는 효과를 주고 있으며, 탭 효과와 애니메이션을 jQuery로 구현해보았습니다.
Mobile 버전과 Pc버전을 분리 제작하여 기기 사이즈별 디자인 작업을하기에 자유로운 장점을 가지고 있는 적응형 웹으로 구성했습니다.

제작기간 3일 참여율 100% 언어 HTML / CSS / JavaScript / jQuery
