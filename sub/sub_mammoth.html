CONCEPT

커피 프랜차이즈 중 하나인 매머드커피를 벤치마킹했습니다. 메인페이지의 풀페이지 효과와 각 섹션의 텍스트 애니메이션 효과를 적용해보았습니다. 서브페이지인 연혁 페이지에서 스크롤시 컨텐츠가 양쪽에서 나오는 효과와 라인이 따라 내려오는 효과도 적용해보았습니다. 반응형으로 제작하여 모든기기에서 컨텐츠가 효과적으로 적용되게 구현했으며, 일관된 class명을 사용하여 유지보수가 간편하게끔 했습니다. 하나의 URL을 사용하여 Pc와 Mobile에서 사용할 수 있기에 검색노출빈도를 높이는 기대를 볼 수 있습니다.

제작기간 5일 참여율 100% 언어 HTML / CSS / JavaScript / jQuery
